# hab-flight-planner

A flight planner for high altitude balloon missions, built in-house by GU Orbit.

This software uses the SondeHub [Tawhiri](https://github.com/projecthorus/tawhiri/) Instance. If you make extensive use of this tool, consider [hosting your own instance](https://github.com/projecthorus/tawhiri/?tab=readme-ov-file#setup) and pointing *hab-flight-planner* to its API endpoint.
